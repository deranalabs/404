# 404Labs Website Specification

## Client Overview
**Name:** 404Labs (NotFoundLabs)
**Focus:** Web3 Infrastructure, specifically Monad Validator operations.
**Vibe:** Modern, clean, professional, monolithic dark mode.

## Core Requirements
1. **Homepage:**
   - Hero section with mission statement.
   - Live Node Status/Stats (Real-time data from Monad Testnet).
   - "About Us" section focusing on infrastructure reliability.
2. **Web3 Integration:**
   - Connect Wallet (Monad Testnet support).
   - Display delegation status if possible.
3. **Design:**
   - Dark mode by default.
   - Minimalist "Terminal-esque" typography.
   - Accents: Monad Purple / Electric Blue.

## Technical Stack
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Wallet: Wagmi / RainbowKit / Reown
- Environment: Monad Testnet

## Success Metrics
- Website loads under 2 seconds.
- Node stats are fetched and displayed correctly.
- Wallet connection is stable on Monad Testnet.
